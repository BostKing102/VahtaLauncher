# Лаунчер для кнопочных телефонов Qin F22

Всем привет, данный лаунчер я разработал для своего друга, который в телефонах полный 0
У лаунчера интуитивно понятный интерфейc, разберется любой кто хоть какое то малейшее представление имеет о кнопочных телефонах

⚠ Сэнсорное управление в главном меню лаунчера не предусмотрено, лаунчер управляется только с кнопок телефона Qin F22 (И других моделей линейки Xiaomi Qin) и использоваться сэнсорным экраном не может ⚠

# Главное меню лаунчера
Главное меню лаунчера полностью доработано, в нем поддерживаются и отображаются функции такие как:
* Обновление времени и даты
* Обновление уровня сигнала связи
* Отображение статуса и название SIM-карты
* Отображение индикатора уровня зарядки и статус зарядки
